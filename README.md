# Pro ID Studio Generator v2.0 🖼️

A sleek, responsive, frontend-only engineering workspace application that dynamically designs, crops, and renders dual-sided corporate identification credentials in real-time.

## 🚀 Features
* **Studio Split Layout:** Organized, tabbed control sidepanel for seamless identity management.
* **Interactive 3D Preview:** Live, fluid 3D card layout engine with realistic perspective flipping animations.
* **Premium Dynamic Themes:** Easily switch styles between *Corporate Blue*, *Midnight Tech*, and *Emerald Executive*.
* **Precision Image Cropping:** Native client-side portrait framing powered by Cropper.js.
* **Flawless Export Pipeline:** High-resolution side-by-side single face PNG downloading driven by html2canvas.

## 🛠️ Built With
* HTML5 (Semantic Structure)
* CSS3 (Flexbox, Grid, 3D Transforms, Custom Variables)
* JavaScript (ES6+, Dynamic DOM Manipulation)
* [JsBarcode](https://github.com/lindell/JsBarcode) (Code-128 Scannable Vector Engine)
* [Cropper.js](https://github.com/fengyuanchen/cropperjs) (Portrait Aspect-Ratio Framing)
* [html2canvas](https://github.com/niklasvh/html2canvas) (High-Resolution Canvas Compilation snapshot)

## 📦 How to Run Locally
1. Clone this repository: `git clone https://github.com/YOUR_USERNAME/REPOSITORY_NAME.git`
2. Open `index.html` directly in your browser or run it via a local environment server extension (like VS Code Live Server).
